# [008 좋은 수 구하기](./src/App008.java)
'좋은 수'란 배열안에서 두 수의 합의로 나올 수 있는 수를 말한다.
예를 들어 [1,2,3,4,6,11] 이라는 배열에서 3은 1,2의 합해서 나올 수 있는 수이고, 4는 1,3으로 나올 수 있는 수이다.
다만 11의 경우 두 수의 합으로 만들어질 수 없는 수이다.
